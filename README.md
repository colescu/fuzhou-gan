# Credits

書法是王羲之與宋高宗（字庫中王羲之缺「話」字）的字。

拼音字體來自[张擎天](https://u-typography.blog/articles/T1.html)，簡單補入了 â ô î û ȳ ý ŷ ỳ 等原字體缺失的撫州話拼音字符。根據原始授權條款，修改後的字體僅作爲本站的一部分使用，不獨立發佈。

音標字體爲 [Charis SIL](https://software.sil.org/charis/) 。
